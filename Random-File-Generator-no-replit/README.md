# Random File Generator

A Node.js + TypeScript project that generates random text files with randomized content and size.

## Features
- Generates files with random names
- Randomized content and spacing
- Configurable file length
- Built using TypeScript and Node.js

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Start (production)

```bash
npm start
```

## Project Structure

```
Random-File-Generator/
├── server/
├── shared/
├── script/
├── package.json
├── tsconfig.json
└── vite.config.ts
```

## Notes
- Ensure Node.js is installed (v18+ recommended)
- Add environment variables in a `.env` file if needed
